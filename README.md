# cyberlite

Typescript sqlite clone

### Usage

| Commands                 | Description                        |
| ------------------------ | ---------------------------------- |
| **insert** <insert_data> | Add a new row                      |
| **select**               | Retrieve all rows                  |
| .help                    | Display list of available commands |
| .clear                   | Clear console                      |
| .exit                    | Terminate program                  |
